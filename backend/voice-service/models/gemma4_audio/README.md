Gemma 4 audio model cache

This directory is the persistent on-disk cache for the optional Gemma speech backend.

Expected environment defaults:

- `GEMMA_MODEL_ID=google/gemma-4-E4B-it`
- `GEMMA_MODEL_CACHE_DIR=/models/gemma4_audio`

The backend uses this directory as the Hugging Face cache for the official model snapshot.
